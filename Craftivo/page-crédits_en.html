Template & Design

This website is based on the Craftivo template created by BootstrapMade, available on bootstrapmade.com. The template has been completely customized and adapted to my needs: full restructuring of sections, modification of the color system, customization of animations, addition of new features, and code optimization to match my visual identity and personal projects.

License: Template used under BootstrapMade license terms
Version: October 2025 update with Bootstrap v5.3.8

Libraries & Frameworks

This website uses several modern libraries and frameworks to ensure an optimal user experience. Bootstrap 5.3.8 forms the basis of the responsive design, allowing perfect adaptation on all devices. AOS (Animate On Scroll) manages smooth animations when scrolling through the page, bringing dynamism to navigation. GLightbox displays portfolio images in gallery mode with zoom. Isotope provides interactive filtering of projects by category. Swiper is used to create modern and touch-friendly carousels. Typed.js animates the presentation text with a typewriter effect. Finally, Waypoints detects scrolling to trigger certain animations at the right time.

Technologies used: Bootstrap 5.3.8, AOS, GLightbox, Isotope, Swiper, Typed.js, Waypoints

Typography

Roboto - Main text: The quick brown fox jumps over the lazy dog

Mulish - Headings: The quick brown fox jumps over the lazy dog

Raleway - Navigation: The quick brown fox jumps over the lazy dog

The fonts used come from Google Fonts and were selected for their complementarity and readability. Roboto was chosen for body text thanks to its exceptional clarity and neutrality, ensuring comfortable reading even on long paragraphs. Mulish brings a modern and dynamic touch to headings with its rounded shapes and contemporary character. Raleway, elegant and refined, is perfect for navigation and interface elements, creating a harmonious visual hierarchy.

External Services

The contact form on this website uses Formspree, an external service that manages message sending without needing to develop a complex backend. This secure and reliable solution processes form submissions and sends them directly by email. The website is hosted and deployed via a professional web hosting service, ensuring optimal availability and fast performance. The domain name is managed by a reliable registrar ensuring service continuity.

Development Tools

To create this website, I used Visual Studio Code as my code editor. It's a free software that makes writing code easier with color coding and suggestions. I used Git and GitHub to save my work and track changes over time. To edit my images, I used Paint 3D which allowed me to improve and adapt my visuals. Finally, I sometimes asked for help from AI assistants like Claude or ChatGPT to fix errors in my code, understand certain functions, or check my spelling and grammar.
